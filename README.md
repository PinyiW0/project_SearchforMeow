## 🖥&ensp;下載專案
>clone 專案
```
git clone https://github.com/PinyiW0/project_SearchforMeow.git
```
>移至專案資料夾
```
cd localFolder/project_SearchforMeow
```
>安裝套件
```
npm install
```
>啟動專案
```
npm dun dev
```
